# AI Story Factory

AI Story Factory is a software project for turning story text into narrated videos with AI-generated visuals, embedded subtitles, music, thumbnails, and metadata.

The first goal is a small MVP:

```text
Input story text
→ Scene JSON
→ Placeholder images
→ Voice
→ Embedded subtitles
→ 60-second MP4
```

The long-term goal is a reusable AI media pipeline for long-form YouTube videos, Shorts, and other story formats.
