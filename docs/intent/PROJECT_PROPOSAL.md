# Project Proposal: AI Story Factory

## Purpose

Build a mostly automated content production system that converts story material into watchable YouTube-style videos with narration, images, subtitles, music, thumbnails, and metadata.

## Primary Goal

Create a side-income content pipeline while keeping the system reusable for multiple formats and niches.

## MVP Scope

The first version will produce a 60-second video from a short story input.

Included in MVP:

- Input story text
- Scene splitting
- Basic narration script
- Placeholder images
- AI or placeholder voice
- Embedded subtitles
- Simple camera motion
- MP4 export

Not included in MVP:

- 2-hour video generation
- Full YouTube upload automation
- Advanced local image generation
- Character consistency system
- Analytics dashboard
- Multi-channel management

## Success Criteria

The MVP is successful when one command can generate a complete 60-second video:

```bash
make demo
```

## Long-Term Vision

Turn the project into a reusable AI Media Factory capable of generating:

- Long-form YouTube videos
- Shorts
- Trailers
- Thumbnails
- Descriptions
- Titles
- Social media clips
